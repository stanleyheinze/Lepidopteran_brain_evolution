Moth_brain_comparative_analysis.Rmd is an R markdown document that calls 
- moths.RData -- a data frame containing the volumetric data for all lepidopteran brains included in the study 
- Kawahara_etal_2019_pruned.tree -- a tree file containing all relevant species.

It contains collection of code that produces plots for figures 1, 5, 6, 7, and 8
of Adden et al.: "The evolution of lepidopteran brain morphology".
